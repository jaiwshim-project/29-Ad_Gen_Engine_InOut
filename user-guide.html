메뉴 구조 이해하기

상단 네비게이션 메뉴

모든 페이지 상단에서 다음 메뉴를 통해 빠르게 이동할 수 있습니다:

  • 홈: 메인 페이지로 이동
  • 생성시작: 새로운 광고 콘텐츠 생성 (등록 업체 선택 또는 신규 입력)
  • 내 프로필: 저장된 프로필로 광고 콘텐츠 재생성
  • 내 결과 보기: 이미 생성한 결과 확인 및 프로필 번호로 검색
  • 카드뉴스 생성기: AI 기반 5장 카드뉴스 자동 생성 (새 탭으로 열림)

홈페이지 메뉴 구성

홈페이지는 두 가지 마케팅 경로와 관리 메뉴로 구성됩니다:

  • 외부 마케팅: 신규 환자 유치를 위한 광고 콘텐츠 생성
  • 내부 마케팅: 기존 환자 관리를 위한 리마인더 메시지 생성

하단 메뉴 카드:

  • 내 프로필: 저장된 치과 프로필 관리 및 재생성
  • 내 결과 보기: 이전에 생성한 광고 콘텐츠 결과 확인
  • 데이터 관리: 저장된 데이터 확인 및 삭제
🚀

시작하기

1

샘플 케이스 체험하기

처음 사용하시는 경우, 두 가지 방법으로 샘플을 확인할 수 있습니다:

  • 홈페이지 중앙 배너: "🎯 샘플 케이스 보기" 버튼 클릭
  • 상단 메뉴: "내 프로필" 메뉴를 통해 저장된 프로필 확인
💡 TIP: 하얀이치과 임영일 원장의 실제 진단 결과를 기반으로 한 샘플로, 어떤 정보가 필요하고 어떤 결과가 나오는지 미리 파악할 수 있습니다.
2

새 케이스 생성하기

홈페이지 또는 상단 메뉴에서 생성을 시작할 수 있습니다:

  • 홈페이지: "외부 마케팅" 버튼 클릭
  • 상단 메뉴: "생성시작" 클릭

정보 입력 페이지에서는 두 가지 방법으로 시작할 수 있습니다:

  • 등록 업체 선택: 상단의 "업체 선택" 섹션에서 기존 업체를 선택하면 모든 정보가 자동 입력
  • 신규 입력: 처음부터 직접 정보를 입력하여 새로운 프로필 생성
🔢 프로필 번호: 새로운 케이스를 생성하면 자동으로 고유한 프로필 번호(예: #0001)가 부여됩니다. 이 번호는 결과 페이지 상단에서 확인할 수 있으며, 나중에 프로필을 쉽게 찾을 수 있도록 도와줍니다.
✍️

정보 입력하기

★

업체 선택 (등록 업체가 있는 경우)

이미 등록된 업체가 있다면, 정보 입력 페이지 최상단의 "업체 선택" 섹션에서 바로 선택할 수 있습니다.

  • 업체 목록: Supabase 클라우드에 저장된 등록 업체 목록이 자동으로 표시됩니다
  • 검색 기능: 일련번호, 병원명, 원장명 필터로 빠르게 검색 가능
  • 자동 입력: 업체를 선택하면 아래 1~6번 모든 항목이 자동으로 채워집니다
  • 수정 및 업데이트: 자동 입력된 정보를 수정한 뒤 "광고 콘텐츠 전략 생성하기" 버튼을 누르면 수정 내용이 Supabase에 자동 업데이트됩니다
💡 TIP: 업체를 선택하면 기본 정보, 매출 단계, 원장 스타일, 신뢰 점수, 철학, 차별화 포인트, 선호 채널, 비밀번호까지 모든 항목이 한번에 자동 입력됩니다. 처음부터 다시 입력할 필요가 없습니다!
1

기본 정보 입력

  • 치과 이름: 귀원의 정식 명칭을 입력합니다
  • 원장 이름: 대표 원장님의 성함을 입력합니다
  • 위치: 치과의 지역 정보를 입력합니다 (예: 서울 강남구)
  • 전문 분야: 주력 진료 분야를 입력합니다 (예: 임플란트, 교정)
2

매출 단계 선택

현재 치과의 매출 상황을 가장 잘 나타내는 단계를 1순위와 2순위로 선택합니다.

  • 1형: 수익 정체·감소
  • 2형: 수익 유지·불안정
  • 3형: 원장 과부하
  • 4형: 수익 안정·확장 준비
  • 5형: 고수익·브랜드형
3

원장 스타일 선택

원장님의 진료 스타일을 가장 잘 나타내는 유형을 1순위와 2순위로 선택합니다.

  • A형: 진료·기술형 - 치료 자체에 집중
  • B형: 학술·연구형 - 최신 기술과 연구에 관심
  • C형: 사람·감성형 - 환자와의 교감 중시
  • D형: 수익·경영형 - 효율적 운영 중시
  • E형: 환자·신뢰형 - 장기적 관계 구축
4

DF-AI 신뢰 점수 입력

0~100점 범위에서 각 항목의 점수를 입력합니다. 점수가 높을수록 해당 영역이 강합니다.

  • 공감: 환자의 감정과 상황을 이해하는 능력
  • 이해: 치료 내용을 명확하게 설명하는 능력
  • 결정 자율성: 환자에게 선택권을 부여하는 정도
  • 가치 전달: 치료의 가치를 효과적으로 전달하는 능력
  • 신뢰 관계: 장기적 신뢰 관계를 구축하는 능력
  • 공간/감성: 치과 환경과 분위기
💡 TIP: 점수는 솔직하게 평가하세요. 낮은 점수도 그에 맞는 최적의 광고 전략을 생성합니다.

🔍 NEW! DF-AI 자가진단 설문 (선택사항)

Step 2.5에서 객관적인 18문항 진단 설문을 제공합니다. 슬라이더를 임의로 선택하는 대신, 구체적인 질문에 답변하면 더 정확한 점수가 자동으로 계산됩니다.

📋 진단 설문 구성:
  • 6개 영역 × 3개 질문 = 총 18문항
  • 각 질문은 5점 척도로 답변 (전혀 아니다 ~ 매우 그렇다)
  • 치과 광고 맥락에 특화된 구체적 행동 기반 질문
  • 예시: "초진 환자의 불안한 표정을 즉시 알아차리고 먼저 안심시킨다"
🎯 사용 방법:
  1. Step 2.5 진단 섹션에서 6개 탭을 순서대로 진행
  2. 각 탭에서 3개 질문에 솔직하게 답변 (자동으로 다음 탭으로 이동)
  3. 18문항 완료 후 "진단 완료" 버튼 클릭
  4. 진단 결과 미리보기에서 6개 점수 + 평균 + 권한 레벨 확인
  5. "결과를 슬라이더에 적용" 버튼으로 자동 입력
  6. 원하면 슬라이더를 수동으로 조정 가능 (유연성 확보)
⚡ 점수 계산 방식:
  • 각 영역의 3개 질문 답변 평균을 0-100 점수로 변환
  • 1점(전혀 아니다) → 0점
  • 2점(아니다) → 25점
  • 3점(보통이다) → 50점
  • 4점(그렇다) → 75점
  • 5점(매우 그렇다) → 100점
💡 진단 vs 슬라이더 직접 입력
  • 진단 설문 (권장): 구체적 질문으로 객관적 평가, 정확도 높음, 5-7분 소요
  • 직접 입력 (선택): "건너뛰고 직접 입력하기" 클릭 시 바로 슬라이더 조절
  • 둘 다 사용: 진단 후 슬라이더 미세 조정 가능 (최고의 정확도)
🔄 재진단 기능: 진단을 완료한 후에도 "다시 진단하기" 버튼으로 언제든 재평가 가능합니다.

📊 평균 신뢰 점수란?

위에서 입력한 6가지 DF-AI 신뢰 점수의 평균값이 자동으로 계산되어 "평균 신뢰 점수"로 표시됩니다.

계산 방식:
평균 신뢰 점수 = (공감 + 이해 + 결정 + 가치 + 신뢰 + 공간) ÷ 6
점수별 의미:
  • 30점 미만: 신뢰 구축이 시급한 단계 → 매우 부드러운 톤 필요
  • 30~44점: 기본적인 신뢰 구축 필요 → 부드러운 톤
  • 45~59점: 보통 수준의 신뢰 → 중간 톤
  • 60~74점: 양호한 신뢰 수준 → 일반적인 광고 톤
  • 75~89점: 높은 신뢰 수준 → 적극적인 톤
  • 90점 이상: 매우 높은 신뢰 → 매우 적극적인 톤
광고 허용 레벨 자동 결정:

평균 신뢰 점수에 따라 광고 콘텐츠 허용 레벨(L0~L5)이 자동으로 결정됩니다:

  • 30점 미만 → L0 (매우 부드러운 톤)
  • 30~44점 → L1 (부드러운 톤)
  • 45~59점 → L2 (중간 톤)
  • 60~74점 → L3 (일반적인 광고 톤) ⭐ 기본값
  • 75~89점 → L4 (적극적인 톤)
  • 90점 이상 → L5 (매우 적극적인 톤)
💡 왜 중요한가?
  • 신뢰 점수가 높을수록: 더 직접적인 마케팅 메시지, 가격/프로모션 강조, 즉각적인 행동 유도(CTA) 가능
  • 신뢰 점수가 낮을수록: 부드럽고 공감적인 메시지, 교육적이고 정보 제공 중심, 관계 구축에 초점
📋 표시 위치: "내 프로필" 및 "내 결과 보기" 페이지의 각 카드에서 "평균 신뢰 점수"로 표시됩니다.
5

추가 정보 입력 (선택)

💡 TIP: 비밀번호는 안전하게 기억해두세요. 분실 시 케이스에 접근할 수 없습니다.

생성된 광고 콘텐츠 샘플

처음 방문하신 분들을 위해 실제 생성되는 광고 콘텐츠의 예시를 보여드립니다. 아래는 "하얀이치과 임영일 원장"의 프로필을 기반으로 생성된 공감형 콘텐츠(중간형) 샘플입니다.

공감형
중간형 (블로그/SNS 긴 글)
프로필 #0001
TITLE

"치과가 무섭다"는 마음, 저희도 이해합니다

BODY

"치과 예약을 미루고 계신가요?"
많은 분들이 치과 방문을 두려워하십니다. 그 마음, 저희가 충분히 이해합니다.

하얀이치과는 환자분의 불안한 마음을 먼저 헤아립니다. 진료실에 들어서는 순간부터 "편안하게 앉으세요"라는 따뜻한 말로 시작합니다.

복잡한 치료 설명도 쉬운 비유로 풀어드리고, 모형과 3D 이미지로 함께 보며 이해를 돕습니다. 급하게 결정하실 필요 없습니다. 집에 가서 가족과 상의하신 후 편하실 때 연락주세요.

치료 후에도 "불편하신 건 없으신가요?" 정기적으로 확인 드립니다. 하얀이치과는 단순히 치료가 아닌, 평생 함께하는 구강 건강 파트너가 되고자 합니다.

CTA (행동 유도)
💬 편안한 상담이 궁금하신가요? 언제든 문의해주세요
HASHTAGS
#강남구치과 #역삼동치과 #하얀이치과 #편안한치과 #임플란트전문 #심미치료 #환자중심진료
💡 이런 콘텐츠가 5가지 유형으로 생성됩니다!
  • 공감형: 환자의 감정과 불안을 이해하는 따뜻한 메시지
  • 이해형: 치료 정보를 쉽고 명확하게 설명
  • 결정형: 환자의 선택권과 자율성을 존중
  • 가치전달형: 장기적 가치와 투자 효과 강조
  • 신뢰형성용: 사후 관리와 약속 이행으로 신뢰 구축
✨ 각 유형별로 3가지 분량 제공
  • 기본형: 짧고 간결 (SNS 피드용, 약 100-150자)
  • 중간형: 적당한 분량 (블로그/SNS 긴 글, 약 300-400자) ⭐ 위 샘플
  • 장문형: 상세한 내용 (홈페이지/브랜드 스토리, 약 600-800자)
✨

결과 확인 및 활용하기

1

프로필 번호 자동 부여 확인

광고 콘텐츠가 생성되면, 결과 페이지 상단에서 자동으로 부여된 프로필 번호를 즉시 확인할 수 있습니다.

  • 표시 위치: 페이지 상단 진단 카드의 "광고 콘텐츠 허용 레벨" 옆에 핑크색 배지로 표시됩니다
  • 번호 형식: "프로필 번호: #0001" 형태로 4자리 숫자
  • 자동 부여: 생성 순서대로 자동으로 부여되며, 첫 번째 프로필은 #0001부터 시작
  • 페이지 로드 시: 부드러운 애니메이션과 함께 나타납니다
💡 TIP: 이 프로필 번호는 케이스가 자동 저장될 때 함께 저장되며, 나중에 "내 결과 보기" 페이지에서 이 번호로 빠르게 검색할 수 있습니다. 번호를 메모해두면 편리합니다!
2

5가지 콘텐츠 유형 탐색

결과 페이지에서 5개의 탭을 통해 다양한 유형의 광고 콘텐츠를 확인할 수 있습니다.

  • 공감형: 환자의 감정과 공감하는 콘텐츠
  • 이해형: 치료 정보를 명확하게 설명하는 콘텐츠
  • 결정형: 환자의 선택권을 강조하는 콘텐츠
  • 가치전달형: 치료의 가치를 강조하는 콘텐츠
  • 신뢰형성용: 장기적 신뢰를 구축하는 콘텐츠
3

콘텐츠 분량 조절

상단의 버튼을 통해 콘텐츠 분량을 조절할 수 있습니다.

  • 기본형: 짧고 간결한 메시지 (SNS에 적합)
  • 중간형: 적당한 분량 (블로그 포스팅에 적합)
  • 장문형: 상세한 내용 (홈페이지, 브랜드 스토리에 적합)
4

광고 허용 레벨 조절

광고의 공격성 정도를 L0~L5까지 조절할 수 있습니다.

  • L0~L1: 매우 부드러운 톤 (신뢰 구축 단계)
  • L2~L3: 중간 톤 (일반적인 광고)
  • L4~L5: 적극적인 톤 (캠페인, 프로모션)

수정 버튼 클릭

2

수정 요청사항 입력

  • "50대 여성 환자를 타겟으로 수정"
  • "할인 이벤트 정보를 추가해주세요"

수정 적용

💡 TIP: 요청사항은 핑크색 박스로 표시되어 실제 사용 시 참고할 수 있습니다.

🔢 프로필 번호 활용 방법

결과 페이지에서 확인한 프로필 번호를 다양하게 활용할 수 있습니다.

✅ 자동 저장 및 번호 부여

  • 결과 페이지가 로드되면 케이스가 자동으로 저장됩니다
  • 생성 순서에 따라 프로필 번호가 자동 부여됩니다 (#0001부터 시작)
  • 같은 케이스를 다시 저장하면 중복 저장되지 않고 기존 번호가 유지됩니다

🔍 번호로 빠른 검색

  • "내 프로필" 페이지 상단의 검색창에 일련번호를 입력
  • 예: "0001" 입력 시 해당 프로필만 즉시 표시
  • 여러 프로필이 있어도 번호로 빠르게 찾을 수 있습니다

📝 수동 저장 시

  • 결과 페이지 하단의 "💾 이 결과 저장하기" 버튼을 클릭하면
  • 알림창에 "케이스가 저장되었습니다! 프로필 번호: #0001" 형태로 번호가 표시됩니다
  • 이미 저장된 케이스는 "이미 저장된 케이스입니다!" 메시지와 함께 기존 번호가 유지됩니다
💡 활용 팁: 프로필 번호를 메모장이나 엑셀에 기록해두면, 여러 치과의 케이스를 관리할 때 매우 편리합니다. 예: "강남치과 - #0001", "서초치과 - #0002"
🔄

새로운 광고 콘텐츠 생성하기

이미 저장된 프로필이 있다면, 두 가지 방법으로 기존 정보를 활용하여 새로운 광고 콘텐츠를 빠르게 생성할 수 있습니다.

방법 1: 정보 입력 페이지에서 직접 선택 (권장)

상단 메뉴의 "생성시작"을 클릭하면 정보 입력 페이지로 이동합니다. 페이지 최상단의 "업체 선택" 섹션에서 등록된 업체를 바로 선택하면 모든 정보가 자동으로 입력됩니다.

  • 검색 필터: 일련번호, 병원명, 원장명으로 빠르게 검색
  • 자동 입력: 선택 즉시 기본 정보부터 비밀번호까지 모든 항목이 채워짐
  • 수정 후 업데이트: 정보를 수정한 뒤 "광고 콘텐츠 전략 생성하기" 버튼을 누르면 수정 내용이 Supabase에 자동 반영

방법 2: 내 프로필 페이지에서 선택

1

내 프로필로 이동

상단 메뉴에서 "내 프로필"을 클릭합니다.

내 프로필 페이지에서는 Supabase에 저장된 모든 프로필 목록을 확인할 수 있으며, 페이지 상단에 검색 기능이 제공됩니다.

2

프로필 카드 찾기

저장된 프로필 목록에서 원하는 치과의 프로필 카드를 찾습니다. 각 카드에는 고유 넘버와 치과 이름이 표시되어 있습니다.

  • 고유 넘버: 각 프로필마다 자동으로 부여된 식별 번호
  • 치과 정보: 치과명, 원장명, 위치 등 기본 정보 표시

🔍 검색 기능 활용하기

프로필이 많아서 찾기 어려운 경우, 페이지 상단의 검색 기능을 사용하면 편리합니다.

  • 일련번호로 검색: 프로필의 고유 번호를 입력 (예: 0001)
  • 병원명으로 검색: 치과 이름의 일부 또는 전체를 입력
  • 원장명으로 검색: 원장님 성함의 일부 또는 전체를 입력
💡 TIP: 검색 입력란에 원하는 조건을 입력하고 "검색" 버튼을 클릭하거나 Enter 키를 누르세요. "초기화" 버튼으로 전체 목록을 다시 볼 수 있습니다.
3

생성하기 버튼 클릭

원하는 프로필 카드에서 "생성하기" 버튼을 클릭합니다.

처음 생성할 때 설정했던 4자리 비밀번호를 입력하는 창이 나타납니다. 비밀번호가 일치하는 경우에만 기본 정보 입력란으로 이동합니다.

⚠️ 주의: 비밀번호를 잊어버린 경우 해당 프로필에 접근할 수 없습니다. 비밀번호를 정확하게 입력해주세요.
4

자동 입력된 정보 확인 및 수정

입력 페이지로 이동하면 저장된 프로필의 모든 정보가 자동으로 입력된 상태로 나타납니다.

  • 치과 이름, 원장 이름, 위치, 전문 분야
  • 매출 단계 선택 (1순위, 2순위)
  • 원장 스타일 선택 (1순위, 2순위)
  • DF-AI 신뢰 점수 (6개 항목)
  • 추가 정보 (치과 철학, 차별화 포인트, 선호 채널, 콘텐츠 유형)
  • 비밀번호
💡 TIP: 자동 입력된 정보를 그대로 사용하거나, 필요한 부분만 수정할 수 있습니다. 수정한 내용은 "광고 콘텐츠 전략 생성하기" 버튼을 누르면 Supabase에 자동으로 업데이트됩니다!
5

콘텐츠 생성

정보를 확인하거나 수정한 후 "광고 콘텐츠 전략 생성하기" 버튼을 클릭하면 새로운 광고 콘텐츠가 생성됩니다.

🎨

AI 카드뉴스 생성하기

주제만 입력하면 AI가 5장의 매력적인 카드뉴스를 자동으로 생성합니다. SNS, 블로그, 홈페이지 등 다양한 채널에서 바로 활용할 수 있는 비주얼 콘텐츠를 만들어 보세요.

★

업체 선택 (등록 업체가 있는 경우)

페이지 최상단에 등록된 업체 목록이 표시됩니다. 업체를 선택하면 업체명, 연락처, 주소, API 키가 자동으로 입력됩니다.

  • 자동 입력: 업체 선택 시 하단의 업체 정보와 API 키가 한번에 채워집니다
  • API 키 자동 로드: 해당 업체에 저장된 API 키가 Supabase에서 자동으로 불러와집니다
💡 TIP: 업체를 선택하면 API 키를 따로 입력할 필요가 없습니다!
1

카드뉴스 페이지로 이동

홈페이지 메뉴 또는 상단 네비게이션에서 AI 카드뉴스 생성기에 접속합니다.

  • 상단 메뉴: 네비게이션 바의 "카드뉴스 생성기" 버튼 클릭 (새 탭으로 열림)
  • 직접 접속: cardnews.html 페이지로 이동
2

API 키 설정

카드뉴스 생성을 위해 Gemini API 키가 필요합니다.

  • 발급처: ai.google.dev에서 무료로 발급
  • 입력: API 키 설정 섹션에서 키를 입력 후 "저장" 클릭
  • 자동 로드: 업체를 선택하면 해당 업체의 API 키가 Supabase에서 자동으로 불러와집니다
  • 보관: 저장된 키는 Supabase 클라우드에 안전하게 보관됩니다
💡 TIP: API 키는 Supabase 클라우드에 저장되므로 다른 기기에서도 업체를 선택하면 자동으로 불러옵니다.
3

도메인(분야) 선택

카드뉴스의 분야를 선택합니다. 8가지 도메인 중 하나를 선택하세요.

  • 병원/의료: 치과, 임플란트, 치아미백, 교정 등
  • 선거/정치: 정치 캠페인, 정책 홍보
  • 교육: 학원, 교육 프로그램, 강좌 안내
  • 부동산: 매물 소개, 분양 안내, 부동산 정보
  • 금융/보험: 금융 상품, 보험 안내, 재테크 정보
  • 뷰티/미용: 피부관리, 시술 소개, 미용 정보
  • 음식/요식업: 메뉴 소개, 맛집 홍보, 신메뉴 안내
  • + 직접입력: 위 목록에 없는 분야를 자유롭게 입력
4

타겟 대상 설정

카드뉴스의 대상 독자를 설정합니다. 연령대와 직업군을 각각 선택하세요.

  • 연령대: 10대 ~ 70대 중 선택
  • 직업군: 직장인, 자영업자, 주부, 학생, 프리랜서, 은퇴자, 전문직 중 선택 (직접 입력도 가능)
💡 TIP: 타겟 대상에 따라 AI가 어조, 용어, 관심사를 자동으로 맞춰 콘텐츠를 생성합니다.
5

주제 및 핵심 메시지 입력

카드뉴스의 주제와 전달하고자 하는 핵심 메시지를 입력합니다.

  • 주제: 카드뉴스의 제목이 될 주제 입력 (예: "치아 미백의 효과와 주의사항")
  • 핵심 메시지: 독자에게 전달하고 싶은 내용을 2~3문장으로 작성
  • 키워드: 관련 키워드를 추가로 선택하거나 직접 입력 (복수 선택 가능)
6

업체 정보 입력

카드뉴스에 포함될 업체 정보를 입력합니다.

  • 업체명: 병원명 또는 사업체명
  • 연락처: 전화번호
  • 주소: 위치 정보

업체 선택 섹션에서 등록 업체를 선택하면 업체 정보가 자동으로 채워집니다. 수동 입력도 가능하며, 입력한 정보는 자동 저장됩니다.

7

카드뉴스 생성 및 결과 확인

"카드뉴스 생성하기" 버튼을 클릭하면 AI가 5장의 카드뉴스를 자동 생성하고, 별도의 결과 페이지로 이동합니다.

  • 1장 (Hook): 시선을 사로잡는 강렬한 제목과 핵심 키워드
  • 2장 (Empathy): 독자의 고민과 상황에 공감하는 스토리
  • 3장 (Problem): 구체적인 문제점과 위험성 제기
  • 4장 (Solution): 해결 방법과 차별화 포인트 제시
  • 5장 (CTA): 행동을 유도하는 마무리 (내원, 문의 등)

결과 페이지에서는 다음 기능을 사용할 수 있습니다:

  • 품질 점수: 전체 점수와 세부 항목별 점수 (Hook 강도, 흐름 일관성, CTA 명확성, 도메인 적합성) 확인
  • 카드 미리보기: 5개 탭으로 각 카드를 크게 미리보기
  • 하단 썸네일: 전체 5장 카드를 한눈에 확인, 클릭 시 해당 카드로 이동
💡 TIP: 각 카드는 고유한 그라데이션 컬러로 디자인되어 시각적 일관성과 차별화를 동시에 제공합니다.
8

카드 텍스트 수정

결과 페이지에서 각 카드의 제목과 내용을 직접 수정할 수 있습니다.

  • 수정 버튼: 카드 미리보기 아래의 "수정" 버튼 클릭
  • 제목/내용 편집: 팝업에서 카드 제목과 본문을 자유롭게 수정
  • 실시간 반영: 저장하면 카드 미리보기와 썸네일에 즉시 반영

자주 묻는 질문 (FAQ)

Q. 생성된 콘텐츠를 상업적으로 사용해도 되나요?
A. 네, 생성된 모든 콘텐츠는 귀원의 마케팅 목적으로 자유롭게 사용하실 수 있습니다.
Q. 데이터는 어디에 저장되나요?
A. 프로필 데이터와 API 키는 Supabase 클라우드에 안전하게 저장됩니다. 브라우저 캐시를 삭제해도 데이터가 유지되며, 다른 기기에서도 동일한 데이터에 접근할 수 있습니다. 광고 콘텐츠 생성 결과는 브라우저의 localStorage에도 함께 저장됩니다.
Q. 여러 치과의 케이스를 관리할 수 있나요?
A. 네, 각 케이스를 저장하면 홈페이지의 "📂 저장된 케이스" 메뉴 또는 상단의 "내 프로필"에서 여러 치과의 케이스를 관리할 수 있습니다. 각 케이스는 고유한 프로필 번호(#0001, #0002 등)가 부여되며, 프로필 번호, 병원명, 원장명으로 빠르게 검색할 수 있습니다.
Q. 생성된 콘텐츠가 마음에 들지 않으면 어떻게 하나요?
A. 콘텐츠 분량과 광고 허용 레벨을 조절하거나, "✏️ 수정" 버튼을 통해 원하는 방향으로 수정 요청을 할 수 있습니다. 또한 다른 콘텐츠 유형 탭을 확인해보세요.
Q. 샘플 케이스는 실제 치과인가요?
A. 네, 하얀이치과 임영일 원장의 실제 진단 결과를 기반으로 한 샘플입니다.
Q. 이전에 저장한 케이스를 어떻게 찾나요?
A. 상단 메뉴의 "내 프로필" 또는 홈페이지의 "📂 저장된 케이스" 카드를 클릭하면 저장된 모든 케이스 목록을 확인할 수 있습니다. 페이지 상단의 검색 기능을 이용하면 프로필 번호, 병원명, 원장명으로 더 빠르게 찾을 수 있습니다.
Q. 상단 네비게이션 메뉴가 안 보여요.
A. 모든 페이지 최상단에 고정된 네비게이션 바가 있습니다. 페이지를 스크롤해도 항상 상단에 표시됩니다. 모바일에서는 메뉴가 축소되어 표시될 수 있습니다.
Q. 프로필 번호는 어떻게 부여되나요?
A. 프로필 번호는 케이스가 생성된 순서대로 자동으로 부여됩니다. 첫 번째로 생성한 프로필은 #0001, 두 번째는 #0002 순으로 부여되며, 결과 페이지 상단의 진단 카드에서 핑크색 배지로 바로 확인할 수 있습니다. 케이스가 자동 저장될 때 함께 저장됩니다.
Q. 내 프로필 번호를 잊어버렸어요.
A. 상단 메뉴의 "내 프로필"로 이동하면 저장된 모든 프로필의 번호를 확인할 수 있습니다. 각 프로필 카드 좌측 상단에 보라색 그라데이션 배지로 "#0001" 형태로 표시됩니다. 프로필 번호를 모르더라도 페이지 상단의 검색 기능에서 병원명이나 원장명으로 검색하면 쉽게 찾을 수 있습니다.
Q. 프로필 번호를 변경할 수 있나요?
A. 아니오, 프로필 번호는 생성 순서에 따라 자동으로 부여되며 변경할 수 없습니다. 이는 각 프로필을 고유하게 식별하기 위한 시스템입니다. 프로필 번호 대신 병원명이나 원장명으로 검색하여 찾는 것을 권장합니다.
Q. 브라우저 캐시를 삭제하면 데이터가 사라지나요?
A. 아니오, 프로필 데이터와 API 키는 Supabase 클라우드에 저장되므로 브라우저 캐시를 삭제해도 유지됩니다. 다른 기기나 브라우저에서도 동일한 데이터에 접근할 수 있습니다.
Q. 카드뉴스 생성에 비용이 드나요?
A. 카드뉴스 생성은 Gemini API를 사용하며, API 키는 ai.google.dev에서 무료로 발급받을 수 있습니다. 무료 사용량 범위 내에서 추가 비용 없이 이용 가능합니다.
Q. 카드뉴스를 병원 외 다른 분야에서도 사용할 수 있나요?
A. 네, 카드뉴스 생성기는 병원/의료, 교육, 부동산, 금융, 뷰티, 음식점, 선거 등 7개 도메인을 지원하며, "직접 입력" 옵션으로 어떤 분야든 자유롭게 활용할 수 있습니다.
Q. 생성된 카드뉴스의 텍스트를 수정할 수 있나요?
A. 네, 결과 페이지에서 각 카드의 제목과 본문을 직접 수정할 수 있습니다. "수정" 버튼을 클릭하면 편집 팝업이 열리며, 수정 내용은 카드 미리보기에 즉시 반영됩니다. 디자인(그라데이션 컬러, 배경 이미지)은 AI가 자동 적용하며, PNG 다운로드 후 별도 도구에서 추가 편집도 가능합니다.
Q. 카드뉴스에서 업체 정보는 어떻게 입력하나요?
A. 카드뉴스 페이지 최상단의 "업체 선택" 섹션에서 등록된 업체를 클릭하면 업체명, 연락처, 주소, API 키가 자동으로 입력됩니다. 등록된 업체가 없는 경우 하단의 업체 정보 입력란에 직접 입력할 수도 있으며, 입력한 정보는 자동 저장됩니다.
